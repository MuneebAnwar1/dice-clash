# 🎲 Dice Clash

**Dice Clash** is a fun and simple two-player dice game played in the terminal. Each player chooses how many times they want to roll a dice (1-6 rolls per round), and the player with the highest total wins the round. Keep playing rounds and see who comes out as the ultimate champion!

## 📦 Features

- 🎯 Roll a standard 6-sided dice with cool unicode dice faces
- 👥 Two-player turn-based gameplay
- 🔁 Multiple rounds with score tracking
- 🏆 Final winner display
- 🛠 Robust input validation

---




## 🚀 Getting Started

### Requirements
- Python 3.x

### How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/dice-clash.git
   cd dice-clash
🎮 How to Play
Each player will be prompted to choose how many times they want to roll the dice (between 1 and 6).

Press Enter each time to roll the dice.

After both players have rolled, the totals are compared.

The player with the higher total wins the round.

The game continues until you choose to stop.

Final scores and the overall winner will be displayed.
